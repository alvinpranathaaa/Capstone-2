# Capstone-2
Capstone Modul 2 Purwadhika, Phyton


Program ini merupakan aplikasi berbasis Python (Terminal) yang digunakan untuk mengelola data pasien rumah sakit menggunakan konsep CRUD (Create, Read, Update, Delete).

Program dilengkapi dengan validasi input untuk memastikan data yang dimasukkan sesuai, seperti:
- ID harus berupa angka (integer)
- Umur harus dalam rentang 0–100 (integer)

Fitur Utama :
- Tambah data pasien
- Tampilkan data pasien
- Update data pasien
- Hapus data pasien
- Search nama pasien
- Sorting data (berdasarkan ID)
- Validasi input (ID & Umur)
